Stripe's docs are the gold standard. Every API endpoint is clear, every error message tells you exactly what went wrong, and the quickstart actually works.

## What I Admire

The thing that strikes me most about Stripe isn't the product — it's the **culture of writing**. Their internal memos are legendary. They treat documentation as a first-class engineering deliverable. That's rare.

Their API design is also exceptional. Idempotency keys, webhook retries, and sensible defaults everywhere. It's clear that engineers who actually use APIs designed this API.

## What I'd Change

The Dashboard UX hasn't kept pace with the API quality. For a company that obsesses over developer experience, the web UI feels like it was designed by a different team with a different philosophy.

I'd push for a tighter feedback loop between the API and the visual layer — especially for debugging failed payments in production.

## Why I'd Want to Work There

The caliber of problems is genuinely hard: distributed money movement, global compliance, fraud at scale. And they seem to give engineers real ownership over those problems.
